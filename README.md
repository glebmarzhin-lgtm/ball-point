# Ball Point 🎯⚙️🎈

Небольшая браузерная игра: натяни **рогатку**, запусти крутящуюся **пилу-шестерёнку** и **лопни все шары**. Пила отскакивает от блоков и стен — чем сильнее натяжение, тем мощнее отскок. А если пила упадёт в **пустоту** — ты её потеряешь!

▶️ **Играть:** https://glebmarzhin-lgtm.github.io/ball-point/

## Как играть
- **Тяни** пилу назад от рогатки и **отпусти**, чтобы выстрелить.
- Сила натяжения = дальность и траектория полёта.
- Пила **отскакивает** от серых блоков и от стен поля.
- Лопни **все шары за один бросок** — уровни собраны так, что это всегда возможно.
- На большинстве уровней без **отскока от блоков** все шары не достать — целься по пунктиру предпросмотра, он учитывает рикошеты.
- Промахнулся или пила упала в **пустоту** — шары просто вернутся, пробуй снова. Попыток сколько угодно.

Работает на компьютере (мышь) и на телефоне (касания).

## Прогресс и уровни
- В игре 20 встроенных уровней + уровни игрока из `levels.js`.
- Прогресс **сохраняется локально** в браузере (`localStorage`): если уже играл, игра сразу продолжается с того уровня, где остановился (стартовое окно не показывается).
- Объекты: блоки-отбойники, пустоты, **повёрнутые и крутящиеся блоки**, **ускорители** (толкают пилу в заданную сторону ~0.5 c).

Уровни хранятся компактной строкой-кодом (base36 фикс. ширины; секции `A`=рогатка, `B`=шары, `K`=блоки, `V`=пустоты, `R`=повёрнутые/крутящиеся блоки, `P`=ускорители). Редактор уровней — отдельный локальный `editor.html`, в опубликованную версию не входит.

## Добавить свои уровни (без правки кода)
1. Открой `editor.html`, собери уровень, нажми **«Копировать»**.
2. Вставь код новой строкой в **`levels.js`** (между обратными кавычками, можно с подписью, напр. `21 ур.: A3w96B...`).
3. Проверь локально (двойной клик по `index.html`), затем опубликуй — уровни работают и локально, и на сайте.

## Обновить сайт
Двойной клик по **`publish.command`** (Mac) — закоммитит и запушит. Или в терминале:

```bash
cd "/Users/gleb/Desktop/Ball point" && git add -A && git commit -m "обновление" && git push
```

Сайт пересоберётся за ~1 минуту.

## Запуск локально
Никаких зависимостей — это чистый HTML/CSS/JS на Canvas. Просто открой `index.html`,
либо подними локальный сервер:

```bash
python3 -m http.server 8123
# затем открой http://localhost:8123
```
