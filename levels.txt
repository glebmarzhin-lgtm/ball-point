# ======================================================================
#  УРОВНИ ИГРОКА для Ball Point  (уровни 21, 22, 23 ...)
# ======================================================================
#
#  Как добавить уровень:
#  1) Открой editor.html, собери уровень, нажми «Копировать».
#  2) Вставь код новой строкой ниже (можно с подписью слева).
#  3) Опубликуй: двойной клик по publish.command
#     (или в терминале — см. README, раздел «Обновить сайт»).
#
#  Правила файла:
#  • один уровень = одна строка;
#  • строки, начинающиеся с #, и пустые строки игнорируются;
#  • подпись слева можно любую — главное, чтобы в строке был код (он начинается с буквы A).
#
#  Примеры (это комментарии, они НЕ добавляются в игру):
#  21 уровень: A3w96B653r8d10al46ct7rezbphj8f
#  22: A3z9dB9f3rf93nlb3rj16vcp6x9q9hl4
#
#  ↓↓↓ ПИШИ СВОИ УРОВНИ НИЖЕ ↓↓↓
